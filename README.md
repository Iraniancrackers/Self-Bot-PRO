# Installation
After installing the dependencies, lets install the bot:
```bash
 git clone https://github.com/CerNerCompany/Self-Bot
 cd Self-Bot
 chmod +x CPM
 ./CPM c --Configing Bot
 ./CPM lc --Login Self
 ./CPM l --Launching Bot
 ---------
 -----Nice Bot-----
 chmod +x Company
 screen Company
 ---------
```
